### 1. 两数之和
```
1. 给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target  的那 两个 整数，并返回它们的数组下标。
2. 你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。
3. 你可以按任意顺序返回答案。
4. 示例：
    1. 输入：nums = [2,7,11,15], target = 9
    2. 输出：[0,1]
    3. 解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。
```
```javaScript
// 哈希 target - nums[i] 取到了则返回
function twoSum(nums, target) {
    const map = new Map();
    for (let i = 0; i < nums.length; i++) {
        const diff = target - nums[i];
        if (map.has(diff)) {
            return [map.get(diff), i];
        }
        map.set(nums[i], i);
    }
    return [];
}
```
### 2. 字母异位词分组
```
1. 给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。
2. 字母异位词 是由重新排列源单词的所有字母得到的一个新单词。
3. 示例：
    1. 输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
    2. 输出: [["bat"],["nat","tan"],["ate","eat","tea"]]
```
```javaScript
// 排序后的key 相同则为异位词
function groupAnagrams(strs) {
    const map = new Map();
    for (let i = 0; i < strs.length; i++) {
        const key = strs[i].split('').sort().join('');
        if (map.has(key)) {
            map.get(key).push(strs[i]);
        } else {
            map.set(key, [strs[i]]);
        }
    }
    return [...map.values()];
}
```
### 3. 最长连续序列
```
1. 给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。
2. 请你设计并实现时间复杂度为 O(n) 的算法解决此问题。
3. 示例：
    1. 输入：nums = [100,4,200,1,3,2]
    2. 输出：4
    3. 解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。
```
```javaScript
// 哈希表 找到前一个的话就跳过，确保是从连续的第一个开始
var longestConsecutive = function(nums) {
    let ans = 0;
    const numSet = new Set(nums);
    numSet.forEach(num => {
        if (numSet.has(num - 1)) {
            return;
        }
        let tempAns = 0;
        let tempNum = num;
        while(numSet.has(tempNum)) {
            tempAns++;
            tempNum++;
        };
        ans = Math.max(tempAns, ans);
    })
    return ans;
};
```
### 4. 移动零
```
1. 给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。
2. 请注意 ，必须在不复制数组的情况下原地对数组进行操作。
3. 示例：
    1. 输入: nums = [0,1,0,3,12]
    2. 输出: [1,3,12,0,0]
```
```javaScript
// 双指针 左指针指向0 右指针指向非0
// 关键点： 指向 0 的指针只有在交换时才会移动
function moveZeroes(nums) {
    let left = 0;
    let right = 0;
    while (right < nums.length) {
        if (nums[right] !== 0) {
            [nums[left], nums[right]] = [nums[right], nums[left]];
            left++;
        }
        right++;
    }
}
```
